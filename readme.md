<h1 align="center">Welcome to my Lofi Music Project</h1>

> Its purpose is to offer a relaxing musical experience around “lofi” music. Interact with the instruments to smoothen your ears.


## Prerequisites

- NodeJS / npm
- Chrome / Firefox / Edge

## Install

```sh
npm install
```

## Usage

```sh
npm run build
```

## Run tests

```sh
npm run dev
```
